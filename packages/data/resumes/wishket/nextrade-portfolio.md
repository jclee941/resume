## 프로젝트 개요

금융 증권거래소 매매체결시스템의 보안 인프라를 설계부터 운영까지 2년간 담당한 프로젝트.

### 1단계: 보안 인프라 구축 (2024.03 ~ 2025.02)

- 고가용성 기반 증권 매매 시스템 보안 아키텍처 설계
- FGCP(FortiGate Clustering Protocol) 기반 Active-Passive HA 구성
- Ansible Role을 활용한 보안 장비 초기 설정 및 정책 배포 표준화
- 망 분리 및 접근통제 정책 설계를 통한 보안 컴플라이언스 준수
- 금융위원회 본인가 심사 기술 대응 및 보안 아키텍처 수립

### 2단계: 보안 운영 (2025.03 ~ 2026.02)

- Splunk ES 및 FortiGate API를 활용한 보안 운영 자동화
- n8n 워크플로우를 연동하여 보안 이벤트 탐지 시 실시간 알림 연동
- FortiManager API를 이용한 방화벽 정책 자동 조회 툴 개발
- 32개 탐지 룰 운영
- 보안 취약점 점검 및 월간 정기 리포트 작성
